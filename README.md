# 📂 Trabalho-2-AEDII

Sistema de manipulação de arquivos desenvolvido para a disciplina de Algoritmos e Estruturas de Dados II, aplicando conceitos de grafos direcionados e ponderados. <br>
Este trabalho é uma continuação do [Trabalho 1 de Algoritmos e Estruturas de Dados II](https://github.com/sarehicy/Trabalho-1-AEDII) e contém todas as funcionalidades desenvolvidas nele com a adição de funcionalidades para a criação e manipulação de um grafo direcional ponderado.
